story

concept

높인다.

design

이미지에 슬라이드 효과를 주어 많은 이미지를 볼 수 있도록 하였다. 베스트 제품은 카테고리를 분류해 나열하였고,
신제품은 프로모션 이미지를 사용하여 홍보효과를 더하였다. 서브 페이지에 들어가지 않고도 제품의 간단한 설명과
컬러를 볼 수 있도록 디자인 하였다.

publishing

coding

웹표준에 기반한 html5 markup 구조를 사용하려 노력했다. 메인 이미지 슬라이드는 bxslider, 신제품 이미지
슬라이드는 slick 플러그인을 이용하였다. 베스트 제품은 setInterval을 이용하여 자동 슬라이드 되도록 하였고
index를 이용해 magin값을 변화시켜 하단 버튼에 따라 이미지가 이동되도록 하였다. 상단 탭을 누르면 같은
index값의 하단 버튼이 클릭 되도록 하여 탭 버튼에 따라서도 이미지가 이동될 수 있도록 하였다. 플러그인을
사용하지 않고 스크립트만을 사용해 구현해보고 싶었는데 잘 되지 않아 아쉬웠다.

develop

태블릿, 모바일 버전으로도 볼 수 있도록 반응형으로 만들었으면 좋았을 것 같다. 슬라이드 효과를 플러그
인을 사용하지 않고 스크립트를 이용하여 구현되도록 시도해 볼 것이다.
